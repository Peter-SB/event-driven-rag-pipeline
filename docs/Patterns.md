
# Patterns

Like a glossary but for design patterns and architectural patterns used in the codebase with explanations, trade-offs, and when/why to use them.

## Publish-Subscribe (Pub/Sub)
Emits events without knowing who consumes them. Decoupled from downstream systems.
Listens for and reacts to events. Can be multiple consumers for the same event.

## Messaging Queue

Producer adds to message queue which is consumed by workers.
